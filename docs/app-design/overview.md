## 一言説明

あなたの「ランダムな」← / → の次の一手を、押す前に読み当てるウェブアプリ。「人は自分が思うほどランダムになれない」を 1〜2 分の体験で突きつける。

## 対象ユーザ

- 面白い・賢い Web を見つけて人にシェアしたいネットユーザ。
- 認知科学・確率・情報理論に軽い興味がある層（専門知識不要）。

## 解決する課題

- 人は自分の予測可能性を過小評価している。それを安全・短時間・無料で体験できる場が少ない。
- 既存の Aaronson Oracle 実装は素朴で、美しく・共有可能で・説明付きの製品がない。

## 提供価値

- **驚き**: 単純操作で Oracle が押下前に当て続ける（MVP 実証: human-like 入力で 64%、random=50%）。
- **理解**: なぜ当たるか（学んだパターン・予測信頼度）を可視化し腑に落とす。
- **拡散**: 「予測されやすさスコア」と共有カードで、ブログ／SNS 紹介が自然に起きる。

## MVP スコープ（P2 で確認済み・本実装には引き継がない）

- ← / → 入力 → 予測 → ヒット表示、ライブ精度、モデル切替。中核体験の成立だけを確認した。

## 本実装スコープ（P5・`app/` で 0 から）

中核ループに、驚き・理解・拡散を足す。

- **コアゲーム**: ← / →（キーボード＋大きなタップ2面）。Oracle は各手の前に予測を確定し、ヒット/ミスを即時開示。
- **ウォームアップ → ロックオン演出**: 最初の数手は「学習中…」、その後「ロックオン」して精度を見せ、"読まれ始めた瞬間"を演出する。
- **ライブ精度チャート**: Oracle 的中率の推移を 50% ベースラインと重ねて表示（軽量 SVG）。
- **予測の中身の可視化**: 現在の文脈（直近 k 手）に対する予測と信頼度、よく当てる文脈の上位（例「L,R,L の後は R 73%」）。
- **予測されやすさスコア**: 単一の共有可能な数値（chance=0%, 完全予測=100%）。
- **共有カード**: スコア＋一言を画像（PNG）化してダウンロード／テキストコピー（完全クライアント）。
- **説明セクション**: Aaronson Oracle / Shannon の読心機 / なぜ人はランダムになれないか（折りたたみ）。
- **モデル設定（上級）**: 予測モデル（n-gram backoff 既定 / Markov-1 / Frequency）と n-gram order の切替、リセット。
- **永続化**: ベストスコア・累計ラウンドを localStorage に保存（アカウントなし）。
- **モバイル対応・アクセシビリティ**: タップ操作、aria-live フィードバック、prefers-reduced-motion 尊重。

## 非スコープ

- アカウント / ログイン / サーバ保存 / サーバ側リーダーボード。
- バックエンド・DB・課金・広告。

- 多言語化（初版は英語 UI。日本語は将来検討）。
- アナリティクス（初版は入れない。必要なら後日 lightweight なものを検討）。

## 成功条件

- 初見ユーザが操作説明なしに 30 秒で「読まれた」と気づく。
- Oracle 的中率が安定して 50% を有意に上回る（実プレイで概ね 60% 以上）。
- 共有カード／スコアが「人に見せたくなる」品質に達している。
- 型 / lint / unit / build が緑で、静的ホスティングにそのまま載る。
